Trigger Mode Demo

Attach the widget to your existing search button

How Trigger Mode Works

Instead of showing a floating button, the widget attaches to an existing element on your page. When that element is clicked, the search modal opens.

<!-- Your existing search button --> <button id="search-trigger">Search</button> <!-- The widget attaches to it --> <script src="https://www.lumify.ai/api/v1/js/lumify-widget.js" data-api-key="YOUR_API_KEY" data-app-id="YOUR_APP_ID" data-mode="trigger" data-trigger-selector="#search-trigger"> </script>
